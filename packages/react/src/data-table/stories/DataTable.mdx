# Data Table

The Data Table component is a powerful and flexible tool for displaying and managing tabular data.
It is designed to handle large datasets efficiently while providing a user-friendly interface for sorting, filtering, and manipulating data.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { DataTable } from "@stewed/react";
```
